Posto Antunes

Posto Antunes é uma Progressive Web Application (PWA) projetada para otimizar a gestão de combustível de uma frota de 200 veículos em uma empresa de transporte escolar. A aplicação conta com um formulário que possui validação tanto no backend quanto no frontend, garantindo a captura e o gerenciamento confiáveis dos dados de abastecimento de combustível.
